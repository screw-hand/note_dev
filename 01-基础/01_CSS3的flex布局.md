定义弹性布局：

```css
/* 块状弹性布局 */
.box {
  display: flex;
}

/* 内联弹性布局 */
.inline {
  display: inline-flex;
}
```

容器属性：

```css
/* 定义主轴方向 */
flex-direction

/* 项目溢出处理 */
flex-wrap

/* 简写属性：主轴方向和溢出处理  */
flex-flow

/* 主轴对齐方式 */
justify-content

/* 侧轴对齐方式 */
align-items

/* 多行侧轴对齐 */
align-content
```

项目属性：

```css
* 排列排序 */
order

/* 放大比例 */
flex-grow

/* 缩小比例 */
flex-shrink

/* 基本尺寸  */
flex-basis

/* 简写属性 */
flex

/* 单项目对齐方式 */
align-self
```
